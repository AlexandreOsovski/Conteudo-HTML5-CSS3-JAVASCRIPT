FORMULÁRIOS

É A TAG MAIS IMPORTANTE DO HTML, SERVE PARA COLOCAR DADOS NO SISTEMA.

Exemplos do formulários:

  • Busca
  • Login
  • Cadastro
  • Formulario de compra
  • Bate-papo

O Formulario é composto pela Tag "form"

FORMULÁRIOS - INPUTS BÁSICOS

ESSES FORMULÁRIOS SÃO USADOS COM MUITA FREQUÊNCIA

  • TEXT
  • PASSWORD
  • RADIO
  • CHECKBOX
  • TEXTAREA
  • FILE

















FORMULÁRIOS - SELECTS

É MUITO PARECIDO COM O RADIO, PERMITE MAIS VALORES



FORMULÁRIOS - BUTTONS E SUBMITS

TODO FORMULÁRIO PRECISA TER OBRIGATORIAMENTE UM DELES, SÃO APARENTEMENTE PARECIDOS, MAS POSSUEM FUNÇÕES DIFERENTES
OS BUTTONS NECESSITAM DE JAVASCRIPT PARA QUE SEJA DETERMINADA A FUNÇÃO DELE
OS SUBMITS SÃO BOTÕES QUE JÁ POSSUEM UM COMPORTAMENTO, OU SEJA, ELES POSSUEM A FUNÇÃO DE "SUBMETER" O SEU FORMULÁRIO A UMA PÁGINA DE DESTINO





FORMULÁRIOS - AUTO-COMPLETE e PLACEHOLDER

AUTO-COMPLETE COMO O NOME JÁ DIZ, ELE AUTO-COMPLETA UMA DETERMINADA ESCRITA QUE VOÇE IRÁ FAZER.
O PLACEHOLDER, ELE DA UMA DICA PARA O USUÁRIO SOBRE O QUE ELE DEVE ESCREVER NO DETERMINADO CAMPO ACESSADO





FORMULÁRIOS - INPUTS ESPECIAIS

SÃO INPUTS QUE NÃO SÃO MUITOS UTILIZADOS, POR ISSO SÃO ESPECIAIS.



  • Color
  • Date
  • Email
  • Range
  • Tel
  • Search

CHEGAMOS NA PARTE MAIS IMPORTANTE DO HTML, A PARTE DOS CONTAINERS, ELAS SÃO TAGS PARA O AGRUPAMENTO DE CONTEÚDOS DENTRO DELAS
